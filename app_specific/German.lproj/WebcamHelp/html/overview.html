Was Sie tun können

Video von Ihrer Webcam ansehen

Schließen Sie Ihre USB webcam an. Wenn sie unterstützt wird, sollten Sie eine "Verbunden mit ..."-Mitteilung in der Statuszeile sehen. Klicken Sie auf das blaue Dreieck in der Werkzeugleiste des Fensters, um das Video zu starten. Voilà! Sie können mit "Kameraeinstellungen" Bildparameter nach Ihrem Belieben einstellen - welche Regler aktiv sind hängt von Ihrer Kamera ab. Mit "Einstellungen merken" können Sie diese Einstellungen als Standardeinstellungen für spätere Programmstarts sichern. Einstellungen für unterschiedliche Kameramodelle werden separat gesichert.

Bilder speichern

Klicken Sie auf das "Kamera mit Blitzlicht"-Icon in der Werkzeugleiste oder wählen Sie "Bild aufnehmen" aus dem "Steuerung"-Menü. Es erscheint ein Fenster mit dem Bild, das sie gewöhnlich speichern können. Im Voreinstellungs-Fenster können Sie das Bildformat für neue Schnappschüsse einstellen.

Filme aufnehmen

Klicken Sie auf das "Kamera mit Filmstreifen"-Icon in der Werkzeugleiste. Die Farbe des Icons ändert sich, um den Aufnahmestatus anzuzeigen. Klicken Sie es erneut, um die Aufnahme zu beenden. Das Video wird als QuickTime-Film abgespeichert und in einem neuen Fenster geöffnet. Im Voreinstellungs-Fenster können Sie das Filmformat und den Speicherort für neue Filme wählen. DOrt können Sie auch einen Zeitfaktor angeben, um Zeitrafferaufnahmen zu machen.

Bilder von der Kamera herunterladen

Wenn Sie eine Kamera anschließen, die als digitale Fotokamera genutzt werden kann und die Bilder enthält, werden Sie gefragt, ob Sie die Bilder herunterladen wollen. Wenn Sie die Frage verneinen, können Sie dies nachholen, indem Sie "Bilder herunterladen..." aus dem "Steuerung"-Menü wählen. Die Bilder werden in der Bildeinstellung heruntergeladen, die auch für Live-Video gilt.

Mehrere Webcams

Wenn Sie mehrere Webcams angeschlossen haben, können Sie zwischen ihnen wechseln, indem Sie "Nächste Kamera" aus dem "Steuerung"-Menü wählen.

Andere Dinge

Sie können Ihre Kamera mit einer Reihe anderer Programme benutzen, wie z.B. BTV, BTV Pro, Oculus, CoolCam, iSpQ VideoChat und viele weitere. Macam funktioniert mit allen Programmen, die QuickTime Videoquellen (manchmal auch VDIG genannt) unterstützen. Um f diese Möglichkeiten zu nutzen, kopieren oder bewegen Sie die Datei "macam.component" in den Ordner "QuickTime" im Ordner "Library" auf der Hauptebene Ihrer Festplatte. Wenn Sie anschließend die QuickTime-Anwendung starten, erscheint jede angeschlossene Kamera als zusätzliche Video-Eingangsquelle. Beachten Sie die Dokumentation des jeweiligen Programms für weitere Details.
